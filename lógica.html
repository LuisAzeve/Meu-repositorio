Arquivo de HTML Modelo OSI Camadas lógicas Camada de aplicação é a camada responsável pela aplicação dos dados, ou seja é a camada final que o usuario usa Camada de apresentação é a camada responsável por traduzir(para a linguagem do computador) as informações dos usuarios resultando na pesquisa Camada de sessão responsável pelo horário inicial e final da sessão (protocolo SIP dentro da camada) Camada de transporte é responsável por separar(segmentar) os dados Camada de rede é responsável por "Empacotar" os dados (o endereço de ip está dentro dessa camada) Camadas fisicas camada de Enlace é responsável por determinar qual a tecnologia que será utilizada para encaminhar os dados (o endereçamento MAC está nessa camada) Camada fisica é responsável por transformar todos os dados em algo fisico
